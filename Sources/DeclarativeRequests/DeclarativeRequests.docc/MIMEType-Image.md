# ``MIMEType/Image``

MIME types in the `image` top-level type.

## Overview

Covers raster, vector, and next-generation image formats:

```swift
MIMEType.Image.png  // "image/png"
MIMEType.Image.svg  // "image/svg+xml"
MIMEType.Image.webp // "image/webp"
```

## Topics

### Raster

- ``png``
- ``jpeg``
- ``gif``
- ``bmp``
- ``tiff``
- ``ico``

### Next-Generation

- ``webp``
- ``heic``
- ``heif``
- ``avif``

### Vector

- ``svg``
