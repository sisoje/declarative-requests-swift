# ``MIMEType/Text``

MIME types in the `text` top-level type.

## Overview

Human-readable text formats — plain text, markup languages, stylesheets,
and structured data:

```swift
MIMEType.Text.html        // "text/html"
MIMEType.Text.eventStream // "text/event-stream"
MIMEType.Text.csv         // "text/csv"
```
